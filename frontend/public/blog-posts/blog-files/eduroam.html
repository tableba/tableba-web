How to connect to eduroam in Arch Linux

This blog post covers over how do connect to eduroam inside arch linux using iwd.

If not already installed install iwd with:

sudo pacman -S iwd

Create eduroam config file

iwctl will not be configured automatically like the other networks, you need to create a config file.

  • Create 
    /var/lib/iwd/eduroam.8021x
    using : 
    mkdir -p /var/lib/iwd && touch /var/lib/iwd/eduroam.8021x
  • Add this content to it:
    [Security]
    
EAP-Method=PEAP
    
EAP-Identity=your_username@your_institution.edu
    
EAP-PEAP-Password=your_password
    
EAP-PEAP-Phase2-Method=MSCHAPV2
    
EAP-PEAP-Phase2-Identity=your_username@your_institution.edu
    
EAP-PEAP-Phase2-Password=your_password
    

    
[Settings]
    
Autoconnect=true

Connect to eduroam

  • Connect to eduroam using the command: 
    iwctl station wlan0 connect eduroam
  • Verify the connection: 
    iwctl station wlan0 show
