Misc git commands

  • Undo the last commit without losing changes: 
    git reset --soft HEAD~1
  • See what changed in any commit: 
    git show <commit-hash>
  • Clean untracked files safely (dry run first): 
    git clean -n
     Actually delete: 
    git clean -f
  • Restore a deleted file from last commit: 
    git restore <filename>
  • List files changed in a branch compared to main: 
    git diff --name-only main..your-branch
  • Add a submodule when cloning a github repository 
    git submodule upadte --init --recursive

Generating a new SSH key

  • Generating a new SSH key

    If your system is relatively recent:

    ssh-keygen -t ed25519 -C "your_email@example.com"

  • Adding SSH key to SSH agent

    Start the agent:

    eval "$(ssh-agent -s)"

    Add you key:

    ssh-add ~/.ssh/id_ed25519

  • Adding SSH key to GitHub

    print the public key (copy it to your clipboard)

    cat ~/.ssh/id_ed25519.pub

    Go to GitHub > Settings > Keys. Then New SSH key > Paste your public key
