Linux commands

This is a non-exhaustive list of handy commands in linux and vim. These are based of an Arch Linux user, if you are using another distrubution some of the commands might not work.

General commands

  • Search for the fastest mirror and append them to 
    /etc/pacman.d/mirrorlist
    (mirrorlist) 
    sudo reflector --latest 20 --protocol https --sort rate --save /etc/pacman.d/mirrorlist
  • Start a http server inside the working directory with python 
    python -m http.server
    It is then accessible in http://localhost:8000/
  • Scan a directory for viruses or attack signatures (using clamav) 
    sudo freshclam
     updates the virus definitions 
    clamscan -r /path/to/folder
     scans the directory
  • Extract a tarball 
    tar -xvf file.tar
     extract 
    tar -xvzf file.tar.gz
     extract gzip 
    tar -xvjf file.tar.bz2
     extract bzip2
  • Copy files using rsync (fancy and faster cp) 
    rsync -avh --progress
    -a: archive mode -v: verbose -h: human readable sizes

Vim commands

  • Delete from the cursor until the end of the line 
    D
  • Repeat the previous command 
    .
  • Recenter cursor in the middle of the screen 
    zz
  • Open file browser 
    :Ex
